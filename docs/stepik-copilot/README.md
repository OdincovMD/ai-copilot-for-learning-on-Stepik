# Stepik Copilot Docs

Документация по продуктовой концепции расширения для Stepik, которое помогает
учащемуся лучше понимать текущий шаг курса и быстрее замечать слабые места в
формулировках, комментариях и собственном понимании материала.

## Что внутри

- [PRODUCT_CONCEPT.md](./PRODUCT_CONCEPT.md) — продуктовая идея, целевая
  аудитория, ценность, границы продукта.
- [MVP_SPEC.md](./MVP_SPEC.md) — что именно входит в MVP, основные сценарии,
  acceptance criteria.
- [ARCHITECTURE.md](./ARCHITECTURE.md) — техническая архитектура расширения и
  серверной части.
- [RISKS_AND_GUARDRAILS.md](./RISKS_AND_GUARDRAILS.md) — риски, ограничения и
  продуктовые защитные меры.
- [ROADMAP.md](./ROADMAP.md) — пошаговый план развития от MVP до следующих
  итераций.
- [BETA_EVAL_PLAN.md](./BETA_EVAL_PLAN.md) — quality gate перед закрытой beta:
  eval-набор, rubric и критерии go / no-go.
- [OPEN_QUESTIONS.md](./OPEN_QUESTIONS.md) — вопросы, которые нужно закрыть до
  старта разработки или публикации.

## Краткое описание продукта

Stepik Copilot — браузерное расширение с AI-сайдбаром, которое анализирует
открытый шаг курса на Stepik и помогает пользователю:

- быстро понять, о чем этот шаг;
- увидеть, на что стоит обратить особое внимание;
- найти типичные точки непонимания по комментариям;
- получить короткий чеклист перед ответом на задание;
- перейти к дополнительному объяснению, не получая готового решения.

## Принципиальная позиция

Продукт не должен превращаться в инструмент для обхода обучения. Основная
ценность — поддержка понимания, а не выдача готовых ответов.
